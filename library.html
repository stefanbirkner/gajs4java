Protocol

The Protocol enum defines four ways of rendering the ga.src assignment of the snippet, which loads the ga.js script.

DECIDE_BY_JAVASCRIPT

This is the default ways to render the assignment. The JavaScript determines whether it loads the ga.js script using http or https. 

ga.src = ('https:' == document.location.protocol ? 'https://ssl' : 'http://www') + '.google-analytics.com/ga.js';

HTTP

The ga.js script is always loaded via http.

ga.src = 'http://www.google-analytics.com/ga.js';

HTTPS

The ga.js script is always loaded via https.

ga.src = 'https://ssl.google-analytics.com/ga.js';

DECIDE_BY_RENDERER

This protocol lets the renderer decide, whether to use HTTP or HTTPS based on the scheme of the HttpServletRequest.

InsertGaJsRenderer

The InsertGaJsRenderer renders the JavaScript snippet which loads the ga.js script. It has two render methods. The writeGaJsInsertStatementToWriter(Writer, Protocol) writes the snippet to the specified writer using the specified protocol. It has not to be used with the DECIDE_BY_RENDERER protocol. The seconde method writeGaJsInsertStatementToWriterUsingRequest(Writer, ServletRequest) uses the request's scheme to decide which of the protocols HTTP and HTTPS to use.
