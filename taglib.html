Taglib

The taglib is a collection of tags, which help you to add Google web tracking to your site.

  • Declaration
  • insertGaJs

Declaration

In order to use this tag library you have to declare it by including a taglib directive before any custom tag is used:

<%@ taglib prefix="gajs" uri="http://stefanbirkner.github.com/gajs4java/tlds/gajs" %>

insertGaJs

The insertGaJs tag renders the snippet, which loads the ga.js script.

The tag 
<gajs:insertGaJs/>
renders the JavaScript code 
(function() {
  var ga = document.createElement('script'); ga.type = 'text/javascript'; ga.async = true;
  ga.src = ('https:' == document.location.protocol ? 'https://ssl' : 'http://www') + '.google-analytics.com/ga.js';
  var s = document.getElementsByTagName('script')[0]; s.parentNode.insertBefore(ga, s);
})();
You may specify a Protocol in order to affect the rendering of the first line. E.g. 
<gajs:insertGaJs protocol="${'DECIDE_BY_RENDERER'}"/>
renders a different third line of the JavaScript code for an http request: 
ga.src = 'http://www.google-analytics.com/ga.js';
